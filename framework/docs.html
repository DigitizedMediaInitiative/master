Getting Started

To start using the framework, the following layout is required for every HTML page you make.

    <!DOCTYPE html>
    <html>
    <head>
    	<meta charset="UTF-8">
    	<title>Page Title</title>
    	
    	<link rel="stylesheet" type="text/css" href="css/master.css">
    	
    	<script src="js/jquery.min.js" type="text/css">
    	<script src="js/scrollUp.min.js" type="text/css">
    	<script src="js/modal.js" type="text/css">
    </head>
    <body>
    	Content Goes Here
    </body>
    </html>

Navigation:

For the navigation bar, copy the syntax for it below.

		Basic
		<div class="nav">
			<ul>
				<li class="title">Your Site Title</li>
				<div class="links">
					<li><a href="#">Link</a></li>
					<li><a href="#">Link</a></li>
					<li><a href="#">Link</a></li>
				</div>
			</ul>
		</div>
		No Title
		<div class="nav">
			<ul>
				<li><a href="#">Link</a></li>
				<li><a href="#">Link</a></li>
				<li><a href="#">Link</a></li>
			</ul>
		</div>
		Title, links on the left
		<div class="nav">
			<ul>
				<li class="title">Your Site Title</li>
				<div class="links links-left">
					<li><a href="#">Link</a></li>
					<li><a href="#">Link</a></li>
					<li><a href="#">Link</a></li>
				</div>
			</ul>
		</div>

Header Box

An example header box

With some sub-text to go along with it

			<div class="head">
				<h1>An example header box</h1>
				<p>With some sub-text to go along with it</p>
			</div>

You can, of course, change the background color. To do that, just add any of the following classes: head-blue, head-orange, head-grey, head-transparent or head-transparent border
